Массивы в Java представляют собой структуру данных, которая позволяет хранить
и обрабатывать набор элементов одного типа.
Массивы имеют фиксированный размер, который определяется при их создании,
и индексируются, начиная с нуля.

- Создание массивов:
Для создания массива в Java используется оператор new, за которым следует тип элемента массива
и размер массива в квадратных скобках.

- Инициализация массивов:
Массивы можно инициализировать при их создании или после создания, указав значения элементов.

- Доступ к элементам массива:
Элементы массива доступны по их индексам, начиная с нуля.

- Длина массива:
Длину массива можно получить с помощью свойства length.

- Многомерные массивы:
Java также поддерживает многомерные массивы, то есть массивы, содержащие другие массивы.

- Ограничения массивов:
Массивы в Java имеют фиксированный размер при создании, и их размер не может изменяться.
Обращение к элементу массива по индексу, выходящему за его границы,
вызывает ошибку ArrayIndexOutOfBoundsException.
При создании массива элементы инициализируются значениями по умолчанию
для соответствующего типа данных (ноль для числовых типов, null для ссылочных типов).

- Методы для работы с массивами:
Java предоставляет ряд методов для работы с массивами через классы из пакета java.util.Arrays.
Эти методы включают в себя сортировку, поиск, заполнение и другие операции над массивами.
