Преобразование и приведение типов (Type Casting) в Java используются
для изменения типа переменной или значения из одного типа в другой.

В Java существует два основных типа преобразования:
- неявное (автоматическое) преобразование
Это автоматическое преобразование типов,
выполняемое компилятором без явного указания программиста.
- явное (ручное) преобразование.
Это преобразование,
которое программист выполняет вручную с использованием операторов приведения типов.
- Приведение типов (Casting)
Приведение типов позволяет изменить тип переменной или значения на другой тип.
Приведение выполняется с использованием операторов приведения типов: (тип)
Приведение типов может потерять данные, если диапазон нового типа меньше,
чем у исходного значения.
- Преобразование при помощи методов:
Некоторые классы в Java предоставляют методы для преобразования данных из одного типа в другой.
Например, Integer.parseInt() для преобразования строки в целое число.
