Double литералы:

- Просто число с десятичной точкой:
double number = 3.14;
- Научная нотация (экспоненциальная форма):
double scientificNumber = 2.0e-3; // 2.0 * 10^(-3)
- Добавление суффикса d или D для явного указания,
что это double (хотя это не обязательно, так как числа по умолчанию считаются double):
double explicitDouble = 123.45d;

Float литералы:
- Добавление суффикса f или F для явного указания, что это float (без суффикса,
числа с плавающей точкой считаются double):
float explicitFloat = 1.23f;

По умолчанию числа с плавающей точкой считаются double
Если литералу принудительно необходимо указать тип double,
то после его значения нужно добавить символ ‘d‘ или ‘D‘.
Если литералу нужно задать тип float, тогда его значение дополняется символом ‘f‘ или ‘F‘.
