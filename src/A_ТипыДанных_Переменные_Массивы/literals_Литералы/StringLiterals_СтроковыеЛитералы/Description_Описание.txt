Представляют собой последовательность символов, заключенных в двойные кавычки (").

Одиночная строка:
String singleString = "Это одиночная строка.";

Пустая строка:
String emptyString = "";

Строка с escape-последовательностями:
String stringWithEscape = "Это строка с символом новой строки:\nи табуляцией:\tи символами \"кавычек\".";

Строка с символами Unicode:
String unicodeString = "Символ евро: \u20AC";

Строка с переводами строк (многострочная строка):
String multiLineString = "Это\nмногострочная\nстрока.";

Строки в Java являются объектами класса java.lang.String, и существует множество методов для работы с ними.
Строковые литералы можно объединять, сравнивать и выполнять различные операции, такие как конкатенация,
поиск подстроки и другие.

Обратите внимание, что строки в Java неизменяемы, что означает,
что после создания строки нельзя изменить ее содержимое.
Если вам нужно внести изменения в строку, вы создаете новую строку с измененным содержанием.
