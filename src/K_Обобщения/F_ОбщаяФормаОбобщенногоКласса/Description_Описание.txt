Синтаксис, представленный в предыдущих примерах программ, может быть обобщен. Ниже показано,
как выглядит синтаксис объявления обобщенного класса.
class имя_класса <список_параметров_типа> { // ...

Обобщенный класс в Java имеет следующую общую форму:
public class GenericClass<T> {
    // Поля и методы класса
}
В этой общей форме GenericClass - это имя обобщенного класса,
и T - это параметр типа, который может быть заменен на конкретный тип данных при создании экземпляра класса.
Параметр типа T может быть использован внутри класса для определения типа переменных,
полей, методов и возвращаемых значений.

А синтаксис объявления ссылки на обобщенный класс и создание его экземпляра полностью
выглядит следующим образом:
имя_класса <список_аргументов_типа> имя_переменной =
 new имя_класса <список_аргументов_типа> (список_аргументов_констант);

 Пример использования обобщенного класса с параметром типа:
 GenericClass<Integer> intGeneric = new GenericClass<>();
 intGeneric.setValue(42);
 int value = intGeneric.getValue();

 GenericClass<String> strGeneric = new GenericClass<>();
 strGeneric.setValue("Hello, World!");
 String text = strGeneric.getValue();
 В этом примере T был заменен на Integer и String для создания двух экземпляров GenericClass
 с разными типами данных.
