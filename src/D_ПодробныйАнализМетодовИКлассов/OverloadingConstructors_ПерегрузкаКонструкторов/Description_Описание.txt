Перегрузка конструкторов (constructor overloading) в Java - это техника,
которая позволяет создавать несколько конструкторов в одном классе с разными параметрами.
Подобно перегрузке методов, компилятор Java различает эти конструкторы на основе количества
и типов параметров.
Это позволяет создавать объекты класса с разными наборами начальных значений.
