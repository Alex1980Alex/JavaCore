В Java ключевое слово final имеет несколько различных использований в зависимости от того,
где оно применяется:
- Финальная переменная (final variable):
Когда final применяется к переменной (полю класса или локальной переменной),
это означает, что значение этой переменной не может быть изменено после его установки.
Это полезно, например, когда вы хотите создать константу.
- Финальный метод (final method):
Если метод в классе объявлен как final, это означает, что этот метод не может быть переопределен
(переопределен) в подклассах.
- Финальный класс (final class):
Если класс объявлен как final, это означает, что этот класс не может быть
расширен (подклассы не могут быть созданы).
// Это не допустимо: class SubClass extends FinalClass { }
- Финальный аргумент (final argument):
Когда final применяется к аргументу метода, это означает,
что значение аргумента не может быть изменено внутри метода.

Использование final может увеличить читаемость кода и помочь в поддержании его надежности,
предотвращая нежелательные изменения значений или структуры классов.
