Перегрузка методов (method overloading) в Java - это механизм,
который позволяет создавать несколько методов с одним и тем же именем в одном классе,
но с разными параметрами.
Компилятор и/или виртуальная машина Java различает эти методы на основе количества
и типов параметров.
При вызове метода выбирается наиболее подходящая версия метода на основе переданных аргументов.
Важно помнить, что перегрузка методов определяется на основе типов и порядка параметров,
но не на основе возвращаемого значения.
