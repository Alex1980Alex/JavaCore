Класс String в Java представляет собой класс, который используется для работы
со строками текста.
Этот класс является одним из самых часто используемых классов в Java,
и он предоставляет множество методов для манипуляции строками.
Вот некоторые основные аспекты и методы класса String:

- Неизменяемость (Immutability):
Строки, созданные с использованием класса String, являются неизменяемыми, что означает,
что их значение не может быть изменено после создания.
Все операции над строками создают новые строки.

- Создание строк:
С использованием литералов: String str = "Пример текста";
С использованием конструктора: String str = new String("Пример текста");

- Конкатенация (Concatenation):
Вы можете объединять строки с помощью оператора + или метода concat():

- Длина строки: Метод length() возвращает длину строки.

- Извлечение символов:
Вы можете извлекать символы из строки по их индексу с помощью метода charAt().

- Сравнение строк:
Для сравнения строк используйте методы equals() или equalsIgnoreCase().

- Поиск подстроки:
Метод indexOf() используется для поиска подстроки в строке.

- Замена подстроки:
Метод replace() заменяет все вхождения подстроки на другую строку.

- Разделение строки:
Метод split() разделяет строку на массив строк на основе заданного разделителя.

- Преобразование к верхнему и нижнему регистру:
Методы toUpperCase() и toLowerCase() преобразуют строку к верхнему или нижнему регистру
соответственно.

Класс String также предоставляет множество других методов для работы с текстом,
таких как substring(), trim(), startsWith(), endsWith(), и другие.
Он является фундаментальным классом для работы со строками в Java
и играет важную роль в большинстве Java-приложений.
