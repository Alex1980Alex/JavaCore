Передача аргументов (параметров) в Java - это важная часть создания и вызова методов.
В Java аргументы могут передаваться методам в нескольких формах:
по значению (копированием значения),
по ссылке (ссылка на объект)
и по значению ссылки (копированием ссылки на объект).
Давайте подробно рассмотрим каждый из этих способов передачи аргументов:
1. Передача аргументов по значению (Pass-by-Value):
В случае примитивных типов данных (int, double, char и т. д.) аргументы передаются методу
по значению.
Это означает, что в метод передается копия значения аргумента, и любые изменения,
внесенные в эту копию, не влияют на оригинальное значение.
2. Передача аргументов по ссылке (Pass-by-Reference):
В Java объекты передаются методам по ссылке, что означает,
что метод имеет доступ к тому же объекту, что и вызывающий код.
Если вы изменяете состояние объекта в методе, эти изменения будут видны и в вызывающем коде.
3. Передача аргументов по значению ссылки (Pass-by-Value of Reference):
Этот случай связан с передачей объектных ссылок, но копируется не сам объект,
а значение ссылки. Это означает, что метод не может изменить саму ссылку на объект,
но может изменить объект, на который она ссылается.
