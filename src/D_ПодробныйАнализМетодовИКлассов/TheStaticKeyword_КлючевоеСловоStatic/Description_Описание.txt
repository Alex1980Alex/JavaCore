Ключевое слово static в Java используется для создания статических членов класса,
которые принадлежат самому классу, а не экземплярам этого класса.
Статические члены класса доступны без создания объекта класса
и совместно используются всеми экземплярами этого класса и другими классами.

Вот несколько основных способов использования ключевого слова static в Java:
- Статические переменные (статические поля):
Статические переменные принадлежат классу, а не экземплярам класса.
Они используются для хранения общей информации для всех объектов этого класса
Вы можете обратиться к статической переменной так: MyClass.staticVariable.
- Статические методы:
Статические методы тоже принадлежат классу и могут быть вызваны без создания объекта класса.
Они часто используются для выполнения операций,
которые не зависят от конкретных экземпляров класса
Вызов статического метода: MyClass.staticMethod().
С методами, объявленными как static, связано несколько ограничений.
• Они могут напрямую вызывать только другие статические методы своего класса.
• Они могут напрямую получать доступ только к статическим переменным своего класса.
• Они никоим образом не могут ссылаться на this или super. (Ключевое
слово super относится к наследованию и описано в следующей главе.)

- Статический блок инициализации:
Вы можете использовать статический блок инициализации для выполнения кода
при первой загрузке класса.
Этот блок выполняется только один раз.
Код в статическом блоке инициализации выполняется автоматически при первом обращении к классу.

Статические члены класса полезны, когда у вас есть общая информация или функциональность,
которую нужно совместно использовать всеми объектами этого класса
или другими частями программы без создания экземпляров класса.
