assert - это ключевое слово в Java, которое используется для включения ассертаций (assertions)
в ваш код. Ассертации представляют собой проверки, которые вы вставляете в программу для того,
чтобы убедиться, что определенные условия выполняются. Если условие ассертации не выполняется,
то программа может завершиться с ошибкой или вывести сообщение об ошибке.
Синтаксис использования assert:
assert booleanExpression;
- booleanExpression - это логическое выражение, которое должно быть истинным (true)
в момент выполнения ассертации.

Вы можете использовать ассертации для проверки предположений о коде, контрактах методов или
инвариантах объектов. Они полезны во время разработки и тестирования для обнаружения ошибок и
некорректных состояний.

Примеры использования assert:
 - Проверка валидности входных данных в методе:
 public void processInput(int value) {
     assert value >= 0 : "Значение должно быть неотрицательным";
     // Далее идет обработка входных данных
 }

 - Проверка инвариантов объекта:
 public class BankAccount {
     private double balance;

     public BankAccount(double initialBalance) {
         assert initialBalance >= 0 : "Начальный баланс должен быть неотрицательным";
         this.balance = initialBalance;
     }

     // Другие методы класса
 }

 - Включение/отключение ассертаций во время выполнения:
 По умолчанию, ассертации выключены в релизной версии Java приложения. Чтобы включить их,
 вы можете использовать флаг командной строки -ea (или -enableassertions) при запуске JVM:
 java -ea YourMainClass
 Чтобы отключить ассертации, вы можете использовать флаг -da (или -disableassertions):
 java -da YourMainClass

 Обратите внимание, что ассертации следует использовать осторожно и только для проверки условий,
 которые не должны произойти в нормальном выполнении программы. Они предназначены для помощи в
 обнаружении и отладке ошибок, но не являются заменой обработки исключений или проверкам
 на нулевые ссылки.
