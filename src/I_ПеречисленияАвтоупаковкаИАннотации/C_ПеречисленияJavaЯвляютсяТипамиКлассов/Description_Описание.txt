В Java перечисления (enums) являются типами классов.
Они представляют собой специальный вид классов, которые ограничены набором предопределенных констант.
При этом каждая константа в перечислении представляет экземпляр этого перечислимого типа.

Внутри перечисления можно определять поля, методы и конструкторы, так же, как и в обычных классах.
Однако, конструкторы перечисления обычно являются приватными,
и экземпляры перечисления создаются автоматически во время инициализации.

Пример перечисления с дополнительными полями и методами:
enum ДниНедели {
    ПОНЕДЕЛЬНИК("Пн"),
    ВТОРНИК("Вт"),
    СРЕДА("Ср"),
    ЧЕТВЕРГ("Чт"),
    ПЯТНИЦА("Пт"),
    СУББОТА("Сб"),
    ВОСКРЕСЕНЬЕ("Вс");

    private String сокращение;

    private ДниНедели(String сокращение) {
        this.сокращение = сокращение;
    }

    public String получитьСокращение() {
        return сокращение;
    }
}
В этом примере каждая константа перечисления ДниНедели имеет свое собственное сокращение,
и есть метод получитьСокращение(), который возвращает сокращение для каждой константы.

Таким образом, перечисления в Java обеспечивают удобный способ определения ограниченных
наборов констант и часто используются для представления списков значений, таких как дни недели, месяцы и т. д.
