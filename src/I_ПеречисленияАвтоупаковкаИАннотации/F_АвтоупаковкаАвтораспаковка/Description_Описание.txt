Автоупаковка (autoboxing) и автораспаковка (unboxing) - это механизмы, предоставляемые Java,
которые позволяют автоматически преобразовывать между примитивными типами данных
и их соответствующими оболочками (wrapper classes) без явного написания кода преобразования.
Эти механизмы сделали работу с оболочками более удобной и читаемой.
- Автоупаковка (Autoboxing):
Это процесс автоматического преобразования примитивного типа данных в его соответствующую оболочку.
Происходит, когда примитивное значение присваивается переменной типа оболочки или передается в метод,
ожидающий аргумент типа оболочки.
Пример автоупаковки:
Integer целое = 42; // Примитивное значение int автоматически упаковывается в Integer
- Автораспаковка (Unboxing):
Это процесс автоматического извлечения значения из оболочки и преобразования его
в соответствующий примитивный тип.
Происходит, когда значение извлекается из объекта оболочки и присваивается переменной
примитивного типа или используется в выражении, ожидающем примитив.
Пример автораспаковки:
Integer целое = 42;
int примитив = целое; // Значение из Integer автоматически распаковывается в int

Автоупаковка и автораспаковка позволяют более гибко работать с примитивами и их оболочками,
особенно в контексте коллекций и обобщенных типов данных, которые требуют использования объектов.
Это также снижает необходимость явного преобразования между примитивами и оболочками,
что делает код более читаемым и уменьшает вероятность ошибок.
