В Java имеется примитивный тип под названием boolean, предназначенный для хранения булевских значений.
Возможных значений только два: true и false.
Значения такого типа возвращаются всеми операциями отношения,например, а<Ь.
Использование типа boolean также обязательно в условныхвыражениях в операторах управления,
таких как if и for.
