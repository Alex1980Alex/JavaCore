В Java существует арифметических операций:

Сложение: + - используется для сложения чисел.
Вычитание: - - используется для вычитания чисел.
Умножение: * - используется для умножения чисел.
Деление: / - используется для деления чисел.
Остаток от деления (модуль): % - используется для вычисления остатка от деления.
Унарный минус: - (унарный оператор) - используется для изменения знака числа на противоположный.
Инкремент ++
Декремент --
Сложение с присваиванием  +=
Вычитание с присваиванием -=
Умножение с присваиванием *=
Деление с присваиванием /=
Деление по модулю с присваиванием %=

Операнды арифметических операций должны и меть числовой тип.
Арифметические операции нельзя использовать с типом boolean, но можно с
типом char, поскольку в Java тип char по существу является подмножеством
типа int.
