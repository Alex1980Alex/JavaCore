Операция присваивания в Java выполняется с использованием оператора =.
Она используется для присвоения значения переменной или элементу массива.

В Java оператор присваивания = используется для присвоения значения правого операнда
(выражения справа от =) левому операнду (переменной или элементу массива слева от =).
Присваивание выполняется справа налево,
то есть результат выражения справа присваивается левому операнду.

- Присваивание значения переменной:
int x = 10; // Присвоить переменной x значение 10
String текст = "Привет, мир!"; // Присвоить переменной текст строковое значение

- Присваивание значения элементу массива:
int[] массив = new int[5];
массив[0] = 1; // Присвоить первому элементу массива значение 1
массив[1] = 2; // Присвоить второму элементу массива значение 2

- Присваивание значения объекту:
Person человек = new Person();
человек.имя = "Иван"; // Присвоить объекту Person имя "Иван"
человек.возраст = 30; // Присвоить объекту Person возраст 30

- Присваивание результатов вычислений:
int a = 5;
int b = 3;
// Присвоить переменной сумма результат сложения a и b (сумма будет равна 8)
int сумма = a + b;
