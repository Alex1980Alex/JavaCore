Интерфейс IntConsumer - это часть Java API и представляет собой функциональный интерфейс,
который определяет метод для принятия одного целочисленного аргумента и выполнения какой-либо
операции над ним.
Этот интерфейс является частью пакета java.util.function, который был добавлен в Java 8
для поддержки функционального программирования и лямбда-выражений.

IntConsumer определяет следующий абстрактный метод:
void accept(int value);

Вы можете реализовать этот метод в соответствии с вашими потребностями,
чтобы выполнить какую-либо операцию, принимая на входе целое число.
Например, вы можете использовать IntConsumer для выполнения некоторых действий
над элементами в потоке целых чисел (например, в массиве или коллекции)
или в других контекстах, когда вам нужно выполнить операции над целыми числами.

Пример использования IntConsumer:
public class IntConsumerExample {
    public static void main(String[] args) {
        IntConsumer printSquare = (int x) -> System.out.println(x * x);
        printSquare.accept(5); // Выведет 25
    }
}
В этом примере создается объект IntConsumer, который принимает целое число,
возводит его в квадрат и выводит результат на экран при вызове метода accept.
