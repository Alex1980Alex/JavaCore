В Java литерал - это фиксированное значение, представленное в исходном коде программы.
Литералы могут быть использованы для представления значений различных типов данных,
таких как целые числа,
числа с плавающей запятой,
символы,
строки
и булевы значения.

Литералы позволяют программистам задавать константные значения напрямую в коде программы.

Вот несколько примеров различных типов литералов в Java:

Целочисленные литералы:
42 - Литерал целого числа.
0xFF - Литерал шестнадцатеричного числа.
123L - Литерал длинного целого числа.

Литералы чисел с плавающей запятой:
3.14 - Литерал числа с плавающей запятой.
2.0e-3 - Литерал числа с плавающей запятой в научной нотации.
123.45f - Литерал числа с плавающей запятой типа float.

Символьные литералы:
'A' - Литерал символа.
'\n' - Литерал символа с использованием escape-последовательности.

Строковые литералы:
"Hello, World!" - Литерал строки.

Логические литералы:
true - Литерал логического значения "истина".
false - Литерал логического значения "ложь".
Null-литерал:

null - Литерал, представляющий отсутствие значения.

Литералы играют важную роль в Java, поскольку они используются для инициализации переменных
и передачи константных значений в коде программы.
