Операторы итерации в Java позволяют вам выполнять повторяющиеся действия в вашей программе.

- Оператор for:
  Оператор for предоставляет более компактный способ выполнения циклов
  с определенным числом итераций.
  for (инициализация; условие; выражение_шага) {
      // тело цикла
  }

- Оператор while:
  Оператор while выполняет блок кода, пока заданное условие истинно.
  while (условие) {
      // тело цикла
  }

- Оператор do-while:
  Оператор do-while похож на while,
  но он всегда выполняет блок кода хотя бы один раз, даже если условие ложно.
  do {
      // тело цикла
  } while (условие);

- Оператор foreach (для итерации по коллекциям):
  Оператор foreach (или расширенный цикл) предназначен для итерации
  по элементам массивов и коллекций.
  for (тип_элемента элемент : коллекция) {
      // тело цикла
  }
