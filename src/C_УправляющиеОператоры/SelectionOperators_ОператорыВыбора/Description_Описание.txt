Java предоставляет несколько операторов выбора,
которые позволяют управлять потоком выполнения программы на основе условий.

Оператор if:
if - это базовый оператор выбора, который позволяет выполнять блок кода,
если заданное условие истинно.
Он может использоваться самостоятельно или совместно с операторами else и else if
для создания более сложных условных конструкций.

Оператор switch:
switch - это оператор выбора, который позволяет выбирать одну из нескольких
альтернативных ветвей выполнения кода на основе значения выражения.
