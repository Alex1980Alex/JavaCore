Ключевое слово finally в Java используется для создания блока кода,
который выполняется независимо от того, произошло исключение или нет.
Блок finally полезен для выполнения кода, который должен быть выполнен всегда,
независимо от того, было ли исключение или нет.
Например, закрытие ресурсов, освобождение памяти или завершение операций важных для программы.
Структура try...catch...finally в Java выглядит следующим образом:
try {
    // Код, который может вызвать исключение
} catch (Exception e) {
    // Обработка исключения
} finally {
    // Код, который выполняется всегда, независимо от того, было ли исключение или нет
}

В этой структуре:
- Блок try содержит код, который может вызвать исключение.
- Блок catch (необязательный) содержит код для обработки исключения, если оно произошло.
- Блок finally (также необязательный) содержит код, который будет выполнен независимо от того,
произошло исключение или нет.
Использование блока finally позволяет гарантировать, что важные операции будут завершены,
даже если возникла ошибка и был выброшен объект исключения.
