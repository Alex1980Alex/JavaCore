Создание многоуровневой иерархии в Java может включать в себя создание классов,
которые наследуются друг от друга, образуя цепочку наследования.
Это позволяет создавать более специализированные классы,
расширяя функциональность базовых классов.

Предположим, у нас есть базовый класс Vehicle,
который представляет общие характеристики всех видов транспорта.
Затем мы создадим два класса, Car и Bicycle, которые будут наследовать от класса Vehicle.
Каждый из этих классов может иметь свои собственные свойства и методы.
