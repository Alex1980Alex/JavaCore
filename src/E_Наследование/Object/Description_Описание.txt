В Java, объект (Object) представляет базовый класс для всех классов.
Это означает, что каждый класс в Java наследует свойства и методы класса Object.
Класс Object определен в пакете java.lang и автоматически импортируется в любой Java-программе.

Вот некоторые из методов, предоставляемых классом Object:
- equals(Object obj):
Метод используется для сравнения объектов на равенство.
По умолчанию этот метод выполняет сравнение по ссылке, то есть два объекта считаются равными,
только если они ссылаются на один и тот же объект.
Многие классы переопределяют этот метод, чтобы сравнивать объекты по их содержимому.
- hashCode():
Этот метод возвращает хеш-код объекта. Хеш-код представляет собой числовое значение,
которое используется, например, при работе с коллекциями, такими как HashMap.
- toString():
Метод возвращает строковое представление объекта. По умолчанию, этот метод возвращает строку,
содержащую имя класса и хеш-код объекта.
Многие классы переопределяют этот метод для предоставления более информативного текстового
представления объекта.
- getClass():
Метод возвращает объект Class, представляющий тип данного объекта.
- wait(), notify(), notifyAll():
Эти методы используются для синхронизации потоков выполнения и работают
в контексте многопоточности.
- clone():
Метод, который позволяет создать копию объекта.
- finalize():
Этот метод вызывается перед удалением объекта сборщиком мусора.

Поскольку класс Object является предком всех классов в Java,
он предоставляет базовую функциональность для работы с объектами.
Важно отметить, что большинство классов переопределяют методы equals(), hashCode()
и toString(), чтобы соответствовать своим потребностям,
и вы можете делать это для собственных классов, если это необходимо.
